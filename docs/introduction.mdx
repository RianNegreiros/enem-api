## Boas-vindas

Bem-vindo à documentação oficial da API do ENEM. Esta API é de código aberto, de uso gratuito e fornece acesso a todas as questões dos exames do ENEM.

Estamos entusiasmados em apresentar a **API v2**, nossa versão mais recente e robusta.

### O que há de novo na v2?

-   **Suporte para "Segunda Aplicação"**: Acesse questões de todas as aplicações de exames.
-   **Novos IDs Únicos**: Um novo campo `id` facilita a identificação e busca de exames específicos.

Recomendamos que todos os novos projetos usem os endpoints da `v2`. Para usuários da nossa API `v1` legada, você pode encontrar a documentação antiga na seção "API v1 Legada".

## Comunidade

Este projeto é possível graças à comunidade. Se você gostaria de contribuir, consulte nosso [guia de contribuição](/contributing).

## Comece por aqui

Este é um projeto criado pela comunidade open-source brasileira, que visa disponibilizar uma API para consulta de provas e questões do ENEM. Todos os dados são públicos e podem ser utilizados sem a necessidade de atribuição de créditos, respeitando a licença [GNU GPL-2.0](https://github.com/yunger7/enem-api/blob/main/LICENSE.md).

<Note>
    Sinta-se livre para [contribuir](/contributing) ou compartilhar seus projetos que utilizam a API, estamos trabalhando em uma seção para divulgá-los em nosso site.
</Note>
